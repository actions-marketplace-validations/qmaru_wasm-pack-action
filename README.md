# `wasm-pack-action`

Install `wasm-pack` by downloading the executable (much faster than `cargo install wasm-pack`, seconds vs minutes).

## Usage

```yaml
- uses: jetli/wasm-pack-action@v0.5.0
  with:
    # Optional version of wasm-pack to install(eg. 'v0.12.1', 'latest')
    version: 'latest'
```
